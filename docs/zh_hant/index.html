什麼是 Bark Sender？

Bark Sender 是一個開源的瀏覽器擴展，讓瀏覽器可以向裝有 Bark App 的 iPhone 推播內容~

無需登入，基於 APNs 推播通知服務，安全可靠。

Bark Sender 瀏覽器擴展

Bark Sender 是執行在瀏覽器中的擴展程式，作為推播流程的傳送端，它可以：

提供推播介面
支援對推播內容加密
支援快速鍵傳送剪貼簿文字
支援網頁右鍵傳送網址或者選中文本
支援點擊圖示一鍵傳送剪貼簿文字
支援解析網頁內容,包括圖片解析並推播
支援頭像,鈴聲，批量推播及 Bark 所有自定義參數
支援地址欄 omnibox 傳送
提供無限的歷史記錄查詢 (保存在瀏覽器 indexedDB 中)

Bark Server 推播伺服器

Bark Server 是開源的推播流程的中轉伺服器，負責處理傳送端的內容並轉發給 APNs 服務。

擴展將推播請求傳送給 Bark Server
Bark Server 預設為 https://api.day.app
可自建部署，確保隱私安全

使用教學

▶ 在 Youtube 上觀看完整影片

或者跟隨以下步驟，快速入門

API URL 用於綁定裝置

API URL 由 https://<伺服器網域>/<裝置Key>/ 組成，你也可以手動輸入或者使用整段 URL 地址,擴展都能識別

在擴展設定中添加你的 iOS 裝置

將剛剛的 API URL 填入 "API URL" 中，設定一個別名來區分裝置，點擊 "添加" 按鈕保存

發起推播內容

點擊瀏覽器擴充功能的圖示，選擇推播文字內容，然後輸入文字，點擊「傳送」按鈕，這樣 iPhone 就可以收到推播了，長按推播或者下拉可以複製到剪貼簿

在 iOS 裝置上安裝 Bark App
並啟用通知權限

關於 Bark [來自 App Store]

簡單調用HTTP介面，即可向你的iPhone傳送推播通知。

Bark支援許多iOS推播的進階特性，例如自定義圖示、通知分組、自定義鈴聲、持續響鈴、重要警告、加密推播等。

Bark不會消耗裝置的電量，其核心功能基於系統推播通知服務與推播擴展，APP本身並不需要執行。

Bark可以非常可靠地保障你的隱私安全。

Bark支援自行部署服務端，可通過Docker、Serverless或Cloudflare Worker部署，bark-server支援絕大多數作業系統，你也可以手動執行它。

APP通過Github Action編譯上傳，確保你使用的APP版本是由開源程式碼編譯，未經任何人（包括作者）修改。

Bark支援自定義密鑰的端到端推播加密，可確保推播內容在傳輸過程中不會被Bark伺服器和蘋果APNs伺服器獲取或洩露。

Bark保存在本地的通知訊息歷史也非常安全。

通過NotificationServiceExtension擴展，在收到推播時將推播訊息保存在本地，不經過其他任何裝置。
