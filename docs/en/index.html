What is Bark Sender?

Bark Sender is an open-source browser extension that allows browsers to push content to iPhones with Bark App installed.

No login required,based on APNs push service,secure and reliable.

Bark Sender Browser Extension

Bark Sender is a browser extension that runs in the browser,serving as the sender in the push process. It can:

Provide push interface
Support encryption of push content
Support keyboard shortcuts to send clipboard text
Support right-click context menu to send URLs or selected text
Support one-click icon to send clipboard text
Support parsing web content,including image parsing and pushing
Support avatars,ringtones,batch pushing and all Bark custom parameters
Support address bar omnibox sending
Provide unlimited history query (saved in browser indexedDB)

Bark Server Push Service

Bark Server is an open-source relay server for the push process,responsible for processing content from the sender and forwarding it to APNs service.

Extension sends push requests to Bark Server
Bark Server defaults to https://api.day.app
Can be self-hosted to ensure privacy and security

Apple Push Notification Service

APNs is Apple's official push notification service,responsible for pushing messages to iOS devices:

Apple's official push service,stable and reliable
Supports push to all Apple devices

iPhone Device + Bark App

The receiver consists of iPhone device and Bark App,responsible for receiving and displaying pushes:

Bark itself is also an open-source APP
Display push content and notifications
Power-saving,no need to run the APP itself
Decrypt encrypted push content

About Bark

Simply call HTTP interface to send push notifications to your iPhone.

Bark supports many advanced iOS push features,such as custom icons,notification grouping,custom ringtones,persistent ringing,important alerts,encrypted push,etc.

Bark does not consume device battery,its core functions are based on system push service and push extensions,the APP itself does not need to run.

Bark can very reliably protect your privacy and security.

Bark supports self-deploying server,can be deployed via Docker,Serverless or Cloudflare Worker,bark-server supports most operating systems,you can also run it manually.

APP is compiled and uploaded through Github Action,ensuring the APP version you use is compiled from open source code,without any modification by anyone (including the author).

Bark supports end-to-end push encryption with custom keys,ensuring push content cannot be obtained or leaked by Bark server and Apple APNs server during transmission.

Bark's locally saved notification message history is also very secure.

Through NotificationServiceExtension,push information is saved locally when push is received,without going through any other devices.
